## Milestone 2 – EDA & Feature Engineering

### Key Insights:
- Content growth peaked around 2018–2019
- Drama dominates Netflix catalog
- US contributes highest content
- Majority content rated TV-MA
- Medium-length movies most common

### Feature Engineering:
- Created year_added feature
- Categorized content length
- Extracted decade feature
- Normalized genres and countries
